Usage

Installation

To use aseconv, first install it using pip:

$ pip install compmat-aseconv

Examples

  • Convert a FHI-aims 1_nacl.in file to VASP poscar file (-t vasp) in Cartesian format (-C) after sorting atom orders by element names (--elsort). Also, add constraint to atoms whose z coordinates are less than 2.5 (--con "z<2.5").

aseconv geo -t vasp --elsort -C --con "z<2.5" 1_nacl.in

Sub-commands

service

Simple aseconv server for faster multi processing.

When processing files multiple times, python loading time could be a bottle neck. This server runs in background with an idle shutdown timeout 30s (default), and receives commands through localhost:17399 (default). Commands can be sent using any TCP clinets, but the following example could be used without installing any other software.

  • An example client command in linux.

MyCOMMANDS needs to be replaced properly. --pwd sets the path of input files, and usually ${PWD}.

    ASEC_SERVER=/dev/tcp/localhost/17399;AFD=793 
    # AFD could be any number (file descriptor).
    exec {AFD}<>${ASEC_SERVER} && echo -n "geo --pwd '${PWD}' MyCOMMANDS [...]" >&${AFD}
    # Open the RW(<>) TCP port and send commands using `echo`.
    ret=1; read -r -u ${AFD} ret; rr=$?; (( rr > 128 )) && ret=1
    # Read a result from server. 0 for success. 
    exec {AFD}<&-   # Port close. Important. 

aseconv service [-h] [--port PORT] [--timeout TIMEOUT] [--stop]

--port

Server TCP port.

Default: 17399

--timeout

Idle shutdown timeout(s).

Default: 30

--stop

Send stop signal to the server.

Default: False

geo

Geometry modification tool using ASE library.

ASE supported formats: abinit-gsr, abinit-in, abinit-out, aims-output, bundletrajectory, castep-castep, castep-cell, castep-geom, castep-md, castep-phonon, cfg, cif, cmdft, cjson, cp2k-dcd, cp2k-restart, crystal, cube, dacapo-text, db, dftb, dlp4, dlp-history, dmol-arc, dmol-car, dmol-incoor, elk, elk-in, eon, eps, espresso-in, espresso-out, exciting, extxyz, findsym, gamess-us-out, gamess-us-in, gamess-us-punch, gaussian- in, gaussian-out, acemolecule-out, acemolecule-input, gen, gif, gpaw- out, gpumd, gpw, gromacs, gromos, html, json, jsv, lammps-dump-text, lammps-dump-binary, lammps-data, magres, mol, mp4, mustem, mysql, netcdftrajectory, nomad-json, nwchem-in, nwchem-out, octopus-in, onetep- out, onetep-in, proteindatabank, png, postgresql, pov, prismatic, py, sys, qbox, res, rmc6f, sdf, siesta-xv, struct, struct_out, traj, turbomole, turbomole-gradient, v-sim, vasp, vasp-out, vasp-xdatcar, vasp-xml, vti, vtu, wout, x3d, xsd, xsf, xtd, xyz

aseconv supported formats: aims, lmp, lmpc, lmpf, rmg

aseconv geo [-h] -t OutputFormat [-i InputFormat] [-C] [-f]
            [--frame start:[stop[:step]]] [-o Outputfile] [--pwd PWD]
            [--sidx SlabIdx] [--noc] [--con "expr of {x-z}"]
            [--strain [FC]ratio{a-cx-z}] [--surface h,k,l,Nlayers]
            [--scale F|Fx,Fy,Fz] [--rotate [C]degree[-]{a-cx-z}[,...]]
            [--align {a-c}{x-z}[,...]] [--elsort] [--zsort]
            [-r na,nb,nc[,opt]] [--vadd vacuum] [--vset setvac]
            [--tr dx,dy,dz] [--sel "expr of {x-z}"] [--torec] [--tomono] [-w]
            [--wp] [--twslab] [--ctrim {''|neighcut|hy}[,...]]
            [--rmlayer l1[,...][,mingap(def:1.9,float)]] [--kp Opt]
            [--ksidx KSlabIdx]
            File/Dir [File/Dir ...]
Positional Arguments
File/Dir

File/Dir of sources.

Named Arguments
-t

Output Format.

-i

Input Format.

-C

Save in Cartesian coordinates (def:fractional).

Default: False

-f, --force

Force write.

Default: False

--frame

Specify frame(s) using python array slicing for multiframe format file such as .xyz.

Default: ':'

-o

Output file name.

--pwd

Working path when using service.

Default: ''

--sidx

-1(auto), 0(non-slab), {1-3}(a-c axis).

Default: -1

--noc

Remove all the constraints.

Default: False

--con

Add additional constraint atoms of the boolean expression. Ex) (z>0)&(z<10).

--strain

Strain structure. F:Fixed volume. C:add lattice constraints (For FHI-aims, xyz axis only).

--surface

Creat a surface of miller indices [h,k,l] with N layers.

--scale

Scale structure. F:Scale factor.

--rotate

Rotate an angle degree around an axis [-]{a-cx-z}. [C]: With cell rotation.

--align

Align a source axis{a-c} to a target axis{x-z}.

--elsort

Sort by elements

Default: False

--zsort

Sort by Z,Y,X.

Default: False

-r

Repeat the cell. n[a-c]:int/float/impr-frac, opt:-1(supercellmode)|tolerance

--vadd

Add vacuum(A) to c axis.

--vset

Set actual vacuum(A) in z axis (auto aligned).

--tr

Translate atoms of d{x-z}(A)

--sel

Select atoms of a boolean expression. Ex) (z>0)&(x>3)

--torec

Hexagonal to rectangular cell.

Default: False

--tomono

Convert a cell to a monoclinic cell(alpha=beta=90).

Default: False

-w

Wrap atom positions into the cell.

Default: False

--wp

Wrap with pretty.

Default: False

--twslab

Translate and wrap a separated slab into middle.

Default: False

--ctrim

Remove dangling carbon atoms [and hydrogenate(hy)]. neighcut (def:3.4)

--rmlayer

Remove layeres along the slabaxis. The layer index(ln) starts from the bottom layer, 0.

--kp

Generate kpath. Options: “”(auto)|kfile(.wkp)|kpstring|ss(show)

--ksidx

Kpath 2D slab index. -1(auto)|0(bulk)|{1-3}(a-c)

Default: -1
