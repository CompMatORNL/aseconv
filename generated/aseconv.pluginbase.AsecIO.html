abstract infos()[source]

Get IO plugin info. Must be implemented in the subclasses.

The typeexts can include multiple dicts. The type is a name for the geometry format such as ‘vasp’, ‘aims’, or etc. The extwithdot is the corresponding extenstion including ‘.’.

Returns:

dict containing

  • ’help’: ‘Help message’

  • ’typeexts’: { ‘type1’: ‘ext1withdot’ , … }
