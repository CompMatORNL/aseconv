aseconv.geoutil.roll_axes

aseconv.geoutil.roll_axes(atom, src, tgt)[source]

Roll axes of the atom image from the src axis to the tgt axis.

Example

Rolling ‘a’ axis to ‘c’ axis results the following axis changes: a->c, b->a, c->b.

>>> roll_axes(atom, 'a', 'c')
Parameters:

  • atom (Atoms) – An atom image.

  • src (str | int) – The source axis. ‘{a-c}’ or 1-3.

  • tgt (str | int) – The tatget axis. ‘{a-c}’ or 1-3.

Returns:

The axes rolled atom image.

Return type:

Atoms
