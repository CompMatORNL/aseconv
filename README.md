# CompMat - aseconv
`aseconv` is a command line based python utility utilizing Atomic Simulation Environment ([ASE]) library to manipulate geometry files. 

## Documents
Please see the [aseconv] document.

## Installation
```sh
pip install compmat-aseconv
```
